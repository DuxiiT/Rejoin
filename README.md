# ReJoin Pro

ReJoin Pro is a professional, standalone Roblox auto-rejoin application designed for high-uptime AFK sessions, multi-instance management, and secure automated logins. Built with a focus on stability and ease of use, it ensures you stay connected even through crashes, kicks, or server restarts.

---

## Key Features

- Monitors Roblox and automatically re-launches the game if it crashes, freezes, or you are kicked.
- Specialized Mutex Silencer logic allows you to run multiple Roblox accounts on the same PC simultaneously.
- Securely store and manage multiple Roblox accounts with automated cookie extraction through a safe browser login.
- Deep integration with popular executors like Cryptic, Xeno, Wave, Ronix, Solara, and more.
- Get real-time notifications on your Discord server whenever a session starts, fails, or re-joins.
- The application automatically manages and downloads its own dependencies (like handle64.exe and chromedriver) at runtime.
- Distributable as a single .exe that runs on any Windows PC without needing Python installed.

---

## Installation

1. Download the latest ReJoin Pro.exe from the [Releases](https://github.com/DuxiiT/Rejoin/releases) page.
2. Run the .exe as Administrator (required for mutex silencing).
3. Enter your License Key and follow the on-screen setup.

---

## Configuration

ReJoin Pro offers a user-friendly CLI menu to manage your setup:

1. License Key: Enter your key from your dashboard.
2. Executor Path: Point the app to your preferred executor's .exe.
3. Autoexec Path: Specify your executor's autoexec folder. The app will automatically place the necessary rejoin.lua script there.
4. Discord Webhook: Paste your Webhook URL to receive remote status updates.
